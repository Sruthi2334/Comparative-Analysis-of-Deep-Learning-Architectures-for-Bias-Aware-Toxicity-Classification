# Bias-Aware Toxicity Classification

A deep learning project for detecting toxic content while addressing demographic bias in NLP models.

# Overview

This project compares two models:

* PH-MSA (CNN + BiGRU + Multi-Head Attention)
* TFRD (TinyBERT with fairness regularization)

# Dataset

Jigsaw Unintended Bias in Toxicity Classification dataset.

# Results

* PH-MSA achieves higher accuracy
* TFRD provides better fairness and macro-F1 balance
